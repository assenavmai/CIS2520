Compilation
************
cd to WhiteVanessaA2 folder. To test the Hash Table ADT 'make table' in the terminal. To test the directory program type 'make testing' into the terminal.

***********************
Running the program(s)
***********************
To compile Hash Table ADT Testing:

Type bin/hash to run the program.
A series of tests will run to test the Hash Table ADT.

To compile the directory program:

Type bin/runTests to run the maze. You do not open the file from the command line. You will be prompted within the program

It will then have a menu display to prompt you to choose an option in this format:

Please choose from one of the following menu options by entering the corresponding number:
1. Load a directory file
2. Add a single entry
3. Print the directory list
4. Reverse Lookup
5. Quit

Please enter your choice now: 

The user then enters their choice from 1 - 5

1. Loads a file from your folder into a linked list and hash table, simply write the name filename.txt/csv or whatever the file extension is
2. Adds a single entry to your loaded directory 
3. Prints the loaded directory
4. User enters a number and it will search the directory for the corresponding person
5. Quit the program
